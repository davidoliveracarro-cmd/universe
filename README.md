# Nébula: Resistencia 9

Videojuego 2D de ciencia ficción en **HTML5 Canvas + JavaScript puro**.

## Ejecutar

```bash
cd /workspace/universe
python3 -m http.server 8000
```

Abre `http://127.0.0.1:8000/`.

> Si ves "Not Found", normalmente el servidor se levantó en otra carpeta: vuelve a ejecutar el comando desde `/workspace/universe`.

## Controles

- **A / D** o **← / →**: mover la nave.
- **ESPACIO**: empezar y disparar.
- **P**: pausa táctica.
- **R**: reiniciar misión.

## Objetivo

Sobrevive a oleadas enemigas, derrota jefes cada 5 oleadas y consigue la máxima puntuación posible.
